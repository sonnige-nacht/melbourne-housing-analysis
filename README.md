# Melbourne Housing Data Analysis

## Project Overview
This project focuses on cleaning and analyzing the Melbourne housing dataset. The data comes from [Kaggle â€“ Melbourne Housing Snapshot](https://www.kaggle.com/datasets/dansbecker/melbourne-housing-snapshot) and includes house prices, number of rooms, land size, location, and other property features.

The goal is to clean the data, handle missing or inconsistent values, and perform statistical and visual analysis to uncover trends and relationships in the Melbourne housing market.

## Key Features
- **Data Cleaning:** Remove missing values, correct invalid entries, and standardize formats  
- **Data Analysis:** Explore price distributions, feature-price correlations, and regional trends  
- **Data Visualization:** Distribution plots, correlation heatmaps, and regional maps
